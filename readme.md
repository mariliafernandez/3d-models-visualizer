# Mesh
Computer Graphics project using OpenGL and Python.


## Available modes
### Transforms
* Rotation (R)
* Translation (T)
* Scale (E)

#### Direction to apply transforms
* x: left/right
* y: up/down
* z: a/d

### Texture
* on/off: 2
