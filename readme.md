# Mesh
Computer Graphics project using OpenGL and Python.

## Run
``` 
python mesh.py obj texture
```
* obj: .obj file path (triangulate faces)
* texture: texture file path (square image)

## Available modes
### Transforms
* Rotation (R)
* Translation (T)
* Scale (E)

#### Direction to apply transforms
* x: left/right
* y: up/down
* z: a/d

### Texture
* on/off: 2
